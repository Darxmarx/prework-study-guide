HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The h1, h2, h3, h4, h5, and h6 elements demonstrate the level of heading a block of text represents.
  • The p element represents a paragraph of text, allowing a user to wrap most of the text on their webpages.
  • The img element contains information about the images displayed on the webpage.
  • The br element creates a line break between blocks of content, forming a line of empty space.

Git

  • git status: checks what branch user is currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: returns to the main branch of a project
  • git add -A: stage all file changes in the current working branch
  • git commit -m "your message here": commits all branch changes to the GitHub repository, with a message stating what changes were made
