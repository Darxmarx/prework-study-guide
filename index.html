Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The h1, h2, h3, h4, h5, and h6 elements demonstrate the level of heading a block of text represents.
  • The p element represents a paragraph of text, allowing a user to wrap most of the text on their webpages.
  • The img element contains information about the images displayed on the webpage.
  • The br element creates a line break between blocks of content, forming a line of empty space.
  • The link element creates a connection to an external file.
  • The rel element establishes the relationship between the current document you're using with the linked document resource.
  • The href element specifies the location of the linked resource relative to the current document's location.

CSS

  • The margin property indicates how much space is taken up outside an element, with a lower value resulting in less space.
  • The padding property indicates how much empty space is added inside an element, resulting in the element being more condensed.
  • The text-align property may be used to align an element to a specific format on a webpage.
  • The img property assigns values to any image element in the project.
  • The display property coordinates where images start.
  • Sizes are measured using px, or pixels.
  • Colors may be assigned with basic semantic references like red or blue, or more specific shades via hexidecimal code.

Git

  • git status: checks what branch user is currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: returns to the main branch of a project
  • git add -A: stage all file changes in the current working branch
  • git commit -m "your message here": commits all branch changes to the GitHub repository, with a message stating what changes were made
