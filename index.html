HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The h1, h2, h3, h4, h5, and h6 elements demonstrate the level of heading a block of text represents.
  • The p element represents a paragraph of text, allowing a user to wrap most of the text on their webpages.
  • The img element contains information about the images displayed on the webpage.
  • The br element creates a line break between blocks of content, forming a line of empty space.
  • The link element creates a connection to an external file.
  • The rel element establishes the relationship between the current document you're using with the linked document resource.
  • The href element specifies the location of the linked resource relative to the current document's location.
  • The script element is located at the closing part of the body element, and functions to link JavaScript to the project.

CSS

  • The margin property indicates how much space is taken up outside an element, with a lower value resulting in less space.
  • The padding property indicates how much empty space is added inside an element, resulting in the element being more condensed.
  • The text-align property may be used to align an element to a specific format on a webpage.
  • The img property assigns values to any image element in the project.
  • The display property coordinates where images start.
  • Sizes are measured using px, or pixels.
  • Colors may be assigned with basic semantic references like red or blue, or more specific shades via hexidecimal code.

Git

  • git status: checks what branch user is currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: returns to the main branch of a project
  • git add -A: stage all file changes in the current working branch
  • git commit -m "your message here": commits all branch changes to the GitHub repository, with a message stating what changes were made

JavaScript

  • Code runs from the first statement at the top of the code down to the last statement at the bottom of the code.
  • The var keyword allows you to declare a variable.
  • An array is a variable that is used to hold a group of data, which is especially handy for performing a single task on multiple pieces of data simultaneously.
  • The location of each item in an array is identified by a unique number called an index.
  • The length of an array is identified by the term length.
  • Arrays are zero-indexed and sequential, so the first item has an index of 0, and subsequent indices will increase by a value of 1 each time.
  • Boolean logic dictates that something's specific value is either true or false.
  • The strict equality operator is documented as === and checks to see if two values are equal, returning a Boolean result afterwards.
  • Strings contain specific text for use elsewhere in the script, and are always within quotation marks.
  • The console.log() method enables outputting a message to the web console by adding an argument.
  • Using if statements, you can create scenarios where code will only execute if certain conditions are met.
  • Conditions that apply to if values are considered truthy.
  • Falsy values include 0, -0, On, "", null, undefined, NaN, and Boolean data with the value of false.
  • Using else if statements allows you to add additional conditions to your code.
  • A for loop uses the predictable pattern of indices in an array to perform a task on all items in an array by enabling a code block to repeat itself over and over again.
  • A for loop is comprised of the starting point for the loop, the condition for the loop to continue, and what allows the array to iterate over each item.
